# Archies Library Phase 1 Website

This is a static demo website for Archies Library Dubai.

## Files
- index.html - main website page
- styles.css - design and layout
- app.js - search, filters, demo forms
- books.json - dummy book catalogue

## Phase 1 limitations
- No real user login
- No payment gateway
- No database
- No admin panel yet
- Contact and book enquiry use mailto links

## Next phases
- Add real inventory spreadsheet/database
- Add Firebase or Supabase login
- Add admin panel
- Add payment gateway
- Add delivery integration
